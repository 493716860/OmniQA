# 项目结构与文件用途

> 说明：本文档以“tree + 用途说明”的方式，快速标注仓库内**主要源码/配置/数据**的职责边界，便于新人上手与面试讲解。

```text
.
├── README.md                         # 项目总览：架构说明、模块关系、运行链路与关键文件导读
├── requirements.txt                  # 后端 Python 依赖清单（Django/DRF/Celery/Playwright 等）
│
├── backend/                          # Django 后端服务（API + 任务系统 + 执行引擎）
│   ├── manage.py                     # Django 管理命令入口（migrate/runserver/自定义命令等）
│   │
│   ├── OmniQA/                       # Django 项目配置包（项目级配置/路由/ASGI/WSGI/Celery）
│   │   ├── __init__.py               # 包初始化（可能包含 MySQL 驱动兼容等初始化逻辑）
│   │   ├── asgi.py                   # ASGI 入口（部署/异步能力扩展的入口文件）
│   │   ├── celery.py                 # Celery 应用初始化入口（Worker 启动时发现任务）
│   │   ├── env.py                    # 极简 .env 读取器（不依赖 python-dotenv；系统环境变量优先）
│   │   ├── settings.py               # Django Settings：DB/Redis/Celery/报告目录/时区/认证等
│   │   ├── urls.py                   # 项目级路由：/api/、/admin/、前端 SPA 托管与刷新兜底
│   │   └── wsgi.py                   # WSGI 入口（传统同步部署入口）
│   │
│   └── api/                          # 业务 API App（领域模型 + REST + 异步任务入口）
│       ├── __init__.py               # Django App 包标记文件
│       ├── admin.py                  # Django Admin 注册（开发排障/数据维护）
│       ├── apps.py                   # Django App 配置
│       ├── authentication.py         # 自定义认证/会话策略（开发/演示环境简化 CSRF 等）
│       ├── models.py                 # 核心领域模型：项目/环境/接口/用例/计划/任务/结果/事件/定时/UI 等
│       ├── serializers.py            # 序列化与规则层：输入输出格式 + 关键业务校验 + 计划选取规则
│       ├── tasks.py                  # Celery 任务定义：run_test_task/run_ui_task/run_due_schedules
│       ├── tests.py                  # 后端自动化测试（关键规则与链路的回归保障）
│       ├── urls.py                   # API 路由表：集中声明前后端交互入口
│       ├── views.py                  # API 控制器层：CRUD/导入/计划预览/任务创建/结果查询/看板聚合/UI
│       │
│       ├── migrations/               # 数据库迁移（记录表结构演进历史）
│       │   ├── __init__.py           # 迁移包标记文件
│       │   ├── 0001_squashed_0011_po_ddt.py   # 初始迁移（包含 UI PO/DDT 等能力的合并迁移）
│       │   ├── 0002_ui_task_batch_dataset.py  # UI 任务与数据集批量执行能力扩展
│       │   ├── 0003_ui_task_celery_task_id.py # UI 任务增加 Celery task id 等字段
│       │   └── 0004_ui_plan_and_exec_task.py  # UI 计划与执行记录（UiPlan/UiExecTask）能力
│       │
│       ├── management/               # Django 自定义管理命令（开发/演示数据/运维辅助）
│       │   ├── __init__.py           # 包标记文件
│       │   └── commands/
│       │       ├── __init__.py       # 命令包标记文件
│       │       ├── import_sample_cases.py     # 导入 Excel 样例用例/资产（便于快速演示接口链路）
│       │       ├── reset_debug_data.py        # 重置并写入一套调试链路数据（展示依赖/Cookie/变量）
│       │       ├── run_due_schedules.py       # 手动扫描到期 Schedule 并触发（开发环境替代 beat）
│       │       ├── seed_demo_data.py          # 写入接口自动化演示数据（项目/环境/接口/用例等）
│       │       └── seed_ui_demo.py            # 写入 UI 自动化演示数据（页面对象/方法/数据集/场景等）
│       │
│       └── services/                  # 服务/执行引擎层（执行、导入、调度等“核心能力”）
│           ├── __init__.py            # 服务层包标记文件
│           ├── curl_importer.py       # cURL 导入引擎：cURL -> 接口定义/用例（快速补齐资产）
│           ├── db_runner.py           # 接口/场景执行引擎：依赖拓扑、变量替换、断言、落库、HTML 报告
│           ├── importer.py            # Excel 导入引擎：解析 Excel -> Project/Env/Module/API/Case
│           ├── scheduler.py           # 定时调度计算：compute_next_run + create_task_from_schedule
│           └── ui_runner.py           # UI 执行引擎（Playwright）：PO/DDT/trace/截图/事件流/结果落库
│
├── frontend/                          # Vue 前端应用（资产维护、计划编排、执行监控、结果展示）
│   ├── index.html                     # 前端宿主页面（Vite 入口 HTML）
│   ├── package.json                   # 前端依赖与脚本（vite dev/build/preview）
│   ├── package-lock.json              # NPM 锁文件（依赖解析结果，不建议手工修改）
│   ├── vite.config.js                 # Vite 构建配置（开发/打包行为）
│   │
│   └── src/                           # 前端源码
│       ├── main.js                    # 应用启动入口：注册 Vue/Pinia/Router/ElementPlus/全局样式
│       ├── App.vue                    # 根组件：提供路由出口
│       │
│       ├── api/                       # 前端 API 通信层（页面不直接拼 URL/写 axios）
│       │   ├── http.js                # Axios 实例封装：超时/withCredentials/统一错误处理
│       │   └── resources.js           # 资源 API 封装：按业务模块组织接口调用（CRUD/动作类 API）
│       │
│       ├── router/
│       │   └── index.js               # 路由定义 + 登录守卫 + history/hash 策略
│       │
│       ├── stores/
│       │   └── auth.js                # Pinia 登录态仓库：登录/登出/用户信息缓存与恢复
│       │
│       ├── layouts/
│       │   └── MainLayout.vue         # 登录后后台布局：菜单/顶部栏/页面承载容器
│       │
│       ├── styles/
│       │   └── global.css             # 全局样式/主题变量/通用容器样式
│       │
│       ├── components/                # 可复用组件
│       │   ├── common/
│       │   │   └── FullScreenDrawer.vue      # 通用全屏抽屉容器（用于复杂表单/编辑器）
│       │   │
│       │   ├── editors/               # 用例/步骤配置编辑器组件（可视化配置的核心）
│       │   │   ├── AssertionEditor.vue       # 断言编辑器：可视化断言与 JSON 双模式
│       │   │   ├── ExtractorEditor.vue       # 变量提取配置编辑器
│       │   │   ├── JsonBodyEditor.vue        # JSON 请求体/配置编辑器
│       │   │   ├── KeyValueEditor.vue        # Header/Query 等键值对编辑器
│       │   │   ├── RequestConfigEditor.vue   # 组合型请求配置编辑器（请求/断言/提取/依赖等）
│       │   │   └── VariablePicker.vue        # 变量选择器：辅助插入 ${变量} 引用
│       │   │
│       │   └── task/                  # “任务中心 / UI 计划”相关交互组件（抽屉/创建/执行）
│       │       ├── UiPlanCreateDrawer.vue     # UI 计划创建抽屉（选择场景/数据集/默认模式等）
│       │       ├── UiScenarioCreateDrawer.vue # UI 场景创建/编辑抽屉（步骤编排入口）
│       │       └── UiTaskRunDrawer.vue        # UI 任务运行抽屉（选择数据集/运行模式并触发）
│       │
│       └── views/                     # 页面层（路由落点）
│           ├── DashboardView.vue      # 质量看板：首页指标/趋势/失败分布/覆盖率等
│           ├── LoginView.vue          # 登录页
│           ├── ProjectsView.vue       # 项目管理页（测试资产的顶层归属）
│           ├── VariablesView.vue      # 项目/环境变量管理页（${var} 引用的来源维护）
│           ├── CookiesView.vue        # 环境 Cookie 管理页（状态型链路能力支撑）
│           ├── ImportView.vue         # 导入页：Excel 导入 + cURL 导入
│           ├── ApiDefinitionsView.vue # 接口定义管理页（path/method/默认 header 等）
│           ├── CasesView.vue          # 接口用例管理页（请求/断言/提取/依赖/标签/套件等）
│           ├── ScenariosView.vue      # 场景用例页（场景与步骤编排）
│           ├── PlansView.vue          # 测试计划页（定义“要跑什么”）
│           ├── SchedulesView.vue      # 定时任务页（简单周期 + Cron）
│           ├── TasksView.vue          # 接口执行任务列表页
│           ├── TaskDetailView.vue     # 接口任务详情页（结果/步骤结果/事件流/报告链接）
│           │
│           ├── UiPagesView.vue        # UI 页面对象/页面方法维护页（PO 模型入口）
│           ├── UiDatasetsView.vue     # UI 数据集维护页（DDT 数据来源）
│           ├── UiScenariosView.vue    # UI 场景编排页（调用页面方法组成场景）
│           ├── UiRunsView.vue         # UI 批量运行入口页（按数据集行生成/触发任务）
│           ├── UiTasksView.vue        # UI 任务列表页
│           ├── UiTaskDetailView.vue   # UI 任务详情页（步骤结果/事件/trace/截图）
│           │
│           ├── TaskCenterView.vue     # 任务中心页面入口（聚合“接口任务 + UI 任务”）
│           └── task_center/           # 任务中心子页（UI 计划/执行记录/调度等）
│               ├── UiPlanView.vue           # UI 计划列表/维护
│               ├── UiScheduleView.vue       # UI 计划/任务的调度视图（若前端提供该入口）
│               └── UiExecRecordsView.vue    # UI 计划执行记录（UiExecTask）列表与详情入口
│
├── data/                              # 示例数据（用于演示/导入）
│   ├── __init__.py                    # 包标记文件（可作为 data 模块占位）
│   └── 控制台首页.xlsx                # Excel 样例用例/资产（可在导入页面直接上传）
│
├── reports/                           # 运行时产物目录（报告/trace/截图；一般不作为源码维护）
│   ├── tasks/                         # 接口/场景执行任务报告输出目录
│   │   └── <task_id>/
│   │       └── html/
│   │           └── index.html         # 单次接口任务 HTML 报告入口
│   └── ui-tasks/                      # UI 任务工件输出目录（Playwright）
│       └── <ui_task_id>/
│           ├── trace.zip              # Playwright trace（回放定位 UI 失败）
│           └── screenshot_step_<id>.png # 失败步骤截图（按步骤结果 id 命名）
│
└── scripts/
    └── dev_backend.py                 # 本地一键启动后端：读 .env -> migrate -> 创建管理员 -> runserver
```
